## What's this?

Creating input/output sample files from Atcoder

## Usage

1. download the target HTML from Atcoder
2. `$ ruby atdecoder.rb the_downloaded_file`

then, you can see some files in the same directory like

- 1_input.txt
- 1_output.txt
- 2_input.txt
- 2_output.txt

```
$ cat 1_input.txt
5 3
10
15
11
14
12

$ cat 1_output.txt
2
```
